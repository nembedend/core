Contributing

Thank you for your interest in contributing.

Before submitting a contribution:

1. Read CLA.md.
2. Ensure your contribution is your own work.
3. Follow the coding style used by the project.
4. Add tests when appropriate.
5. Update documentation if required.

By submitting a contribution you agree to the terms
described in CLA.md.

Pull requests that do not comply with the CLA may be rejected.
